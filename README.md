# Email Spam Detection using Machine Learning

## Overview

This project was completed as part of the **VortexTech AI & ML Internship – Week 2**. The objective was to build a binary classification model that predicts whether an email is **Spam** or **Ham (Not Spam)** using machine learning.

## Dataset

* Dataset: Email Spam Dataset
* Task: Binary Classification (Spam = 1, Ham = 0)

## Features
* Loaded and preprocessed the dataset
* Encoded the target variable
* Split the data into training and testing sets
* Converted text into numerical features using **TF-IDF Vectorizer**
* Trained a **Multinomial Naive Bayes** classifier
* Evaluated the model using Accuracy, Precision, Recall, and F1-Score
* Visualized model performance with charts and a confusion matrix
* Saved the trained model and vectorizer using Joblib

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Joblib
* Google Colab

## How to Run

1. Clone this repository.
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Upload the **email spam.xlsx** dataset.
4. Run all notebook cells to train and evaluate the model.
